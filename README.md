## Intro

Heart of the Spire is a Laravel-based web application social media centred around roguelike card game 'Slay the Spire.' The web app allows users to post and comment about cards, builds and strategies.

## Demo

Watch a full showcase of the web app in action in this <a href="https://youtu.be/UfH4KBaTIDg?si=p6oAZBsGQIKcr1Lj">youtube video!</a>

## Features

- User profiles: Register to create new accounts and Log in with passwords.
- Seeders and Factories: Quickly generate content to explore the site with automatically generated posts and comments.
- Profile Customisation: Users can set a bio, favourite class and profile picture.
- Admin Roles: Admins have elevated permissions to moderate posts and comments.
- Posts: Users can create posts and attach multiple game cards to each post.
- Comments: Users can comment on posts, with real-time updates via AJAX (no page refresh).
- Clean, high-quality UI: Built with Tailwind CSS for a modern and responsive design.
- Pagination: Easily browse through lots of posts and cards.
- Database and Relationships: Efficient database essentials followed and one-to-one, one-to-many, and many-to-many relationships implemented.
- MVC & Laravel: Full utilisation of Laravelâ€™s MVC structure for maintainable and scalable code.
- Data Validation: All input is validated so that the user cannot break the web app and has a seamless experience.

## Experience and Lessons Learnt

- Building a Social Media: Learned all the components and essentials that make up every social media platform and how to connect everything effectively.
- AJAX Integration: Implemented real-time comment posting to improve user experience without full page reloads. This is tricky but provides the user a more seamless experience.
- UI Design: Experimented frequently with different layouts and tools to try and find the best and most appealing designs for the web app and accessibility too
- Full Stack Web App Workflow: Used Laravel, Blade templates, Models, Views, Controllers and Relationships. This was my first ever experience at a full stack project working simultaneously with the front end and back end to work together seamlessly but I see it is dificult and will be better next time

## Set Up

To access the project and explore it yourself please follow this set up.

### Requirements

- Laravel 10+
- Docker
- Artisan
- Modern web browser

### Steps

- Clone the repository
- Install all PHP and Node dependencies needed for a Laravel project
- Run migrations and seeders
- Build front end assets
- Put the sail up
- Open http://localhost in your browser
