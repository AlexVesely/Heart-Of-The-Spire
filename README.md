## Intro

Heart of the Spire is a Laravel-based web application social media platform centred around the roguelike card game <i>Slay the Spire</i>. The web app allows users to create accounts and to write posts and comments about cards, builds and strategies.

## Demo

Watch a full showcase of the web app in this <a href="https://youtu.be/UfH4KBaTIDg?si=p6oAZBsGQIKcr1Lj">youtube video!</a>

## Features

- <b>User profiles:</b> Register accounts and log in with passwords.
- <b>Seeders and Factories:</b> Populate the app quickly with sample posts, comments, and users.
- <b>Profile Customisation:</b> Profiles can set a bio, favourite class and profile picture.
- <b>Admin Roles:</b> Admins have elevated permissions to moderate posts and comments.
- <b>Posts:</b> Users can create posts and attach multiple game cards to each post.
- <b>Comments:</b> Users can comment on posts, with real-time updates via AJAX (no page refreshes).
- <b>Modern UI:</b> Clean, responsive front-end designed with Tailwind CSS.
- <b>Pagination:</b> Easy browsing of posts and cards.
- <b>Data Validation:</b> User inputs validated to prevent errors and ensure a smooth experience.
- <b>Databases:</b> Databases for each model created and following proper database convention.
- <b>Database Relationships:</b> One-to-one, one-to-many, and many-to-many relationships implemented properly.
- <b>MVC & Laravel:</b> Full use of Laravelâ€™s MVC architecture for maintainable, scalable code.

## Experience and Lessons Learnt

- <b>Building a Social Media:</b> Learned how posts, comments, profiles, and interactions connect in a standard social media application.
- <b>AJAX Integration:</b> Learned how to implement content posting without page reloads, improving the user experience.
- <b>UI Design:</b> Learned how to develop more advanced front-end layouts using Tailwind CSS and Blade templates.
- <b>Full Stack Web App Workflow:</b> First ever experience at a full stack project working simultaneously with Laravel back-end logic and Blade front-end displays.

## Set Up

To access the project and explore it yourself please follow this set up.

### Requirements

- Laravel 10+
- Docker
- Artisan
- Modern web browser

### Steps

- Clone the repository
- Install all PHP and Node dependencies needed for a Laravel project.
- Run migrations and seeders.
- Build front end assets.
- Put the sail up.
- Open localhost in your browser.
